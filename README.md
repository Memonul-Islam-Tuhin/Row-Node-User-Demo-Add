# Row-Node-User-Demo-Add

A simple JavaScript demo project that demonstrates managing users with a node-like structure.

## Project Structure

- `public/` – Contains static files for the project.
- `admin.js` – Main JavaScript file handling user operations.
- `users.json` – Sample data file containing user information.
- `package.json` – Project configuration and dependencies.

## Features

- Add, read, and manage users.
- Simple JSON-based data storage.
- Node-like structure for organizing rows/users.
